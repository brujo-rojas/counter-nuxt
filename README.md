# Counter Nuxt

Gestión de items con Nuxt 4 y Vuex.

## Stack

- Nuxt 4.0.3
- Vuex 4.0.2
- TypeScript
- SCSS

## Desarrollo

```bash
npm install
npm run dev
```

## Funcionalidades

- CRUD de items
- Filtros y búsqueda
- Persistencia localStorage/sessionStorage
- Límites: 20 items máximo, valores 0-100
